## 작성 지침

1. **키워드 활용법**
   - 키워드를 억지로 나열하지 말고 자연스럽게 연결
   - 키워드들 사이의 공통점이나 연결고리를 찾아 하나의 이야기로 엮기
   - 모든 키워드가 반드시 명시적으로 드러날 필요 없음

2. **연령대 고려사항**
   - 노안을 고려한 적절한 문단 나누기
   - 지나치게 트렌디한 신조어/외래어 자제
   - 경험에서 우러나온 진정성
   - 교훈보다는 공감과 위로

3. **절대 피해야 할 것**
   - 가르치려는 태도
   - 과거 미화 일변도 또는 과도한 회상
   - "요즘 젊은이들은..." 식 불평
   - 지나치게 무겁거나 우울한 전개/결말
   - 키워드가 부정적이라고 해서 글 전체를 부정적으로 쓰는 것

4. **효과적인 요소**
   - 구체적인 감각 묘사 (냄새, 소리, 촉감)
   - 계절감, 시간대
   - **현재 진행형** 일상의 순간 (과거 아닌 지금)
   - 밝고 따뜻한 여운 (쓸쓸하거나 무거운 여운 지양)
