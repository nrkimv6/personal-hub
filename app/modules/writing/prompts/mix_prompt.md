반드시 JSON 형식으로 응답해주세요. 다른 설명 없이 JSON만 출력하세요.

## ⚠️ 중요 지침 (최우선 적용)

예시문이 회상적이거나 부정적인 톤이더라도:
- **생성할 글의 70% 이상은 밝고 긍정적인 톤**을 유지할 것
- 예시문의 쓸쓸하거나 그리운 감정은 도입부에만 간결하게 반영
- **현재의 기쁨, 감사, 희망**을 중심으로 새로운 글 전개
- 과거 회상은 최소화하고, 현재 진행형 내용 우선

**중요**: 예시문의 톤을 그대로 따라하지 말고, **밝고 따뜻한 새로운 글**을 만들 것

## 소재(topic) 추출 지침

- 단일 명사(사랑, 마음)가 아닌 **2-5단어의 구체적 소재**
- 좋은 예: "어머니의 된장찌개", "40년 된 밥솥", "첫눈 오던 날 아침"
- 나쁜 예: "사랑", "추억", "가족"

## 글 작성 지침

- 3개 예시문의 요소가 모두 녹아들되, **완전히 새로운 밝은 글**
- 50-70대 독자에게 **감동/힐링/희망** 중 하나 전달
- 과거 미화/회상 일변도 피할 것 → **현재의 기쁨** 중심
- 구체적인 감각 묘사 포함 (냄새, 소리, 촉감)
- 예시문이 어두워도 생성할 글은 **밝고 따뜻하게**
