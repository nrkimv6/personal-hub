# 모니터링 시스템 API 서버 PRD

## 1. 개요
웹 페이지 모니터링 시스템의 API 서버로, 기존 모니터링 시스템을 마이그레이션하여 확장성과 관리 용이성을 개선합니다.

## 2. 기술 스택
- FastAPI: API 서버 프레임워크
- Sqlite
- Pydantic: 데이터 검증
- Playwright: 웹 페이지 모니터링

## 8. 특별 기능 구현 상세

### 8.1 날짜 기반 스케줄링 시스템
- URL의 날짜 파라미터를 기반으로 모니터링 간격을 자동 계산하는 기능
- 미래 날짜에 따른 우선순위 기반 모니터링:
  - 오늘/이미 지난 날짜: 2-7초 간격으로 빠른 모니터링
  - 내일: 3-10초 간격으로 모니터링
  - 1-7일 이내: 20-50초 간격으로 중간 주기 모니터링 
  - 7일 이상: 200-300초 간격으로 낮은 주기 모니터링
- 사용자 정의 간격 설정 지원:
  - 자동 계산된 간격 외에도 사용자가 직접 간격 설정 가능
  - 사용자 정의 간격 설정 시 자동 재계산 방지 기능
- 날짜나 URL 변경 시 간격 자동 재계산 (사용자 정의 아닌 경우)

### 8.2 콘텐츠 파싱 및 알림 시스템
- HTML 콘텐츠에서 유용한 정보 추출하여 알림에 포함
- parse_time_and_stock(): 버튼 텍스트에서 시간과 매수 정보 추출
- parse_page_info(): 페이지에서 제목, 시간, 매수 등 종합 정보 추출
- 다양한 형식의 웹페이지 대응 가능한 유연한 파싱 로직
- 추출된 정보를 사용자 친화적인 형태로 알림 메시지에 통합
